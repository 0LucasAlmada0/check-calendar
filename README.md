# Check Calendar

Check Calendar e um app simples feito em React Native para ajudar pessoas a organizarem seus checks diarios.

Com ele, o usuário pode marcar as tarefas do dia conforme forem concluídas, mantendo uma visão clara do que ainda precisa ser feito e do que já foi finalizado.

## Funcionalidades

- Criar e acompanhar tarefas do dia.
- Marcar tarefas como concluídas.
- Manter os checks salvos no dispositivo.
- Apoiar a rotina diaria com uma lista simples e objetiva.

## Tecnologias

- React Native
- AsyncStorage

## Como o app ajuda

O Check Calendar facilita a organização dos checks diarios ao centralizar as tarefas em um único lugar. Isso ajuda o usuário a manter foco, acompanhar o progresso do dia e reduzir o risco de esquecer atividades importantes.

## Objetivo

Oferecer uma experiência prática para quem deseja transformar tarefas do dia em pequenos checks acompanhaveis, tornando a rotina mais organizada e visual.
