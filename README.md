# ✅ Check Calendar

Check Calendar é um app simples feito em React Native para ajudar pessoas a organizarem seus checks diários. Com ele, o usuário pode marcar as tarefas do dia conforme forem concluídas, mantendo uma visão clara do que ainda precisa ser feito e do que já foi finalizado.

## ✨ Funcionalidades
- 📝 Criar e acompanhar tarefas do dia.
- ✅ Marcar tarefas como concluídas.
- 💾 Manter os checks salvos no dispositivo.
- 🎯 Apoiar a rotina diária com uma lista simples e objetiva.

## 🛠️ Tecnologias
- ⚛️ React Native
- 📦 AsyncStorage

## 🌟 Como o app ajuda
O Check Calendar facilita a organização dos checks diários ao centralizar as tarefas em um único lugar. Isso ajuda o usuário a manter foco, acompanhar o progresso do dia e reduzir o risco de esquecer atividades importantes.

## 🎯 Objetivo
Oferecer uma experiência prática para quem deseja transformar tarefas do dia em pequenos checks acompanháveis, tornando a rotina mais organizada e visual.

## ⚙️ Instalação
1. Clone este repositório:  
   `git clone https://github.com/0LucasAlmada0/check-calendar`
   
2. Navegue até a pasta do projeto:  
   `cd check-calendar`

3. Instale as dependências:  
   `npm install` (ou `yarn install`)

4. Inicie o projeto (usando Expo):  
   `npx expo start`

5. Escaneie o QR code com o aplicativo Expo Go no seu dispositivo ou escolha rodar no emulador.
